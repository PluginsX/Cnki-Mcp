## ⚠️ 注意事项

### 1. 首次运行

- 需要手动完成验证码
- 浏览器窗口会自动弹出
- 测试会在验证码处暂停等待

### 2. 依赖要求

```bash
pip install mcp playwright pydantic
playwright install chromium
```

### 3. 网络要求

- 需要能访问知网（kns.cnki.net）
- 稳定的网络连接

### 4. 时间预估

- 完整测试：1-2分钟（不含验证码等待）
- 快速测试：5-10秒
