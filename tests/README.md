## ⚠️ 常见问题

### 1. 导入错误

**问题**：`ModuleNotFoundError: No module named 'mcp'`

**解决**：
```bash
pip install mcp
```

### 2. Playwright 未安装

**问题**：`playwright._impl._api_types.Error: Executable doesn't exist`

**解决**：
```bash
playwright install chromium
```

### 3. 验证码超时

**问题**：测试在验证码处卡住

**解决**：
- 手动完成验证码
- 等待页面加载完成
- 测试会自动继续

### 4. 浏览器未启动

**问题**：看不到浏览器窗口

**解决**：
- 检查 `browser.py` 中 `headless=False`
- 确保没有其他程序占用浏览器

### 5. 连接超时

**问题**：`TimeoutError: Server did not respond`

**解决**：
- 增加等待时间
- 检查服务是否正常启动
- 查看 stderr 输出

## 🔍 调试技巧

### 1. 查看详细日志

修改测试脚本，添加日志输出：

```python
import logging
logging.basicConfig(level=logging.DEBUG)
```

### 2. 单独测试某个功能

注释掉其他测试，只运行需要的：

```python
# await self.test_cnki_search()
# await self.test_cnki_get_status()
await self.test_cnki_set_page_size()  # 只测试这个
```

### 3. 增加等待时间

如果网络较慢，增加等待时间：

```python
await asyncio.sleep(5)  # 从2秒改为5秒
```

### 4. 手动测试

使用 Python 交互式环境：

```python
import asyncio
from test_complete import test_all_tools

asyncio.run(test_all_tools())
```
